# Інформаційна система управління графіків та змін персоналу
Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра  
([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)).  
Керівник роботи – **Подунова Анастасія Олександрівна**.

## Завдання до дипломної роботи
Розроблений програмний продукт має являти собою десктопну систему управління
графіками змін працівників на базі JavaFX та SQLite.  
Система забезпечує автоматизацію обліку робочого часу, управління персоналом,
формування графіків змін, розрахунок відпрацьованих годин, аналітику та експорт даних.
