# Розробка додатка "Інформаційна система управління графіків та змін персоналу"
Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра  
([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)).  
Керівник роботи – **Подунова Анастасія Олександрівна**.

## Завдання до дипломної роботи
розроблений програмний продукт має являти собою десктопну систему управління графіками змін працівників на базі JavaFX та SQLite. Система має забезпечувати автоматизацію обліку робочого часу, управління персоналом та експорт даних. Функціонал включає формування графіків, розрахунок відпрацьованих годин та аналітику. Інтерфейс повинен бути інтуїтивно зрозумілим з табличним відображенням даних, кольоровим кодуванням типів змін та підтримкою гарячих клавіш для швидкої навігації.
## Використані технології
- мова програмування Java 17;
- середовище розробки IntelliJ IDEA;
- графічний інтерфейс користувача — JavaFX;
- база даних — SQLite.
