## Мета роботи
Розробка десктопного програмного застосунку для автоматизації управління
графіками змін персоналу з використанням JavaFX та SQLite.

## Основні можливості
- формування та редагування графіків змін;
- облік робочого часу працівників;
- збереження даних у базі SQLite;
- експорт графіків у формат Excel;
- зручний табличний інтерфейс.

## Використані технології
- Java 17  
- IntelliJ IDEA  
- JavaFX  
- SQLite
